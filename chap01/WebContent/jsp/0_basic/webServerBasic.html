# 루프백 주소

  • 자기 컴퓨터를 가리키는 주소
  • 127.0.0.1 : 공유기까지 갔다가 돌아온다.
  • localhost : 공유기까지 가지도 않는다.

# IP주소 : 포트번호로 서버를 찾아간다.

  • 웹 브라우저에서는 80번(http) 포트와 443번(https) 포트는 생략이 가능
  • 우리가 사용 하는 테스트용 포트는 반드시 IP주소 뒤에 입력해야 한다.
  • 포트번호 뒤에 나오는 '/' 이후의 경로가 서버내에서 사용하는 경로가 된다.

# 우리서버는 왜 네이버처럼 이름으로 들어올 수 없을까?

  • DNS서버에 우리 IP주소가 등록되어있지 않기 때문이다.
  • DNS서버(Domain Name Service) : 보기싫은 IP주소를 보기좋은 영어 이름으로 바꿔주는 서비스
  • 영어로된 URL을 입력하면 바로 해당 서버를 찾아가는 것이 아니라 DNS서버에서 IP주소를 받은 뒤 찾아가게 된다.
